## 单元目标

- 开启 SAP SuccessFactors HXM 之旅，了解可供您使用的资源
- 识别 What's New Viewer 中的功能
- 描述实例
- 描述配置
- 配置访问管理
- 描述实例如何与配置相关
- 登录 SAP SuccessFactors
- 介绍 SAP SuccessFactors 主页
- 定义人员和操作搜索

- 识别管理员任务
- 解决用户登录问题
